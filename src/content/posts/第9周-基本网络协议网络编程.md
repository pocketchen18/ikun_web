## 一、网络基础概念

### 1.1 网络通信模型（OSI七层模型）

```
应用层    ← HTTP, FTP, SMTP, DNS
表示层    ← 数据加密、压缩
会话层    ← 建立、管理会话
传输层    ← TCP, UDP
网络层    ← IP, ICMP
数据链路层 ← Ethernet, WiFi
物理层    ← 网线、光纤、无线电波
```

**简化版（TCP/IP四层）：**
```
应用层    ← HTTP, FTP
传输层    ← TCP, UDP
网络层    ← IP
网络接口层 ← Ethernet
```

### 1.2 IP地址和端口

```
IP地址：192.168.1.100    ← 找到哪台电脑
端口号：8080             ← 找到哪个程序

就像：
IP地址 = 大楼地址
端口号 = 房间号
```

**常见端口：**
| 端口 | 服务 |
|------|------|
| 80 | HTTP |
| 443 | HTTPS |
| 22 | SSH |
| 21 | FTP |
| 3306 | MySQL |

## 二、TCP协议

### 2.1 TCP特点

- **可靠**：保证数据到达，不丢失
- **面向连接**：通信前先建立连接（三次握手）
- **有序**：数据按顺序到达
- **面向字节流**：没有消息边界

**比喻：** TCP就像打电话
- 先拨号建立连接
- 说话保证对方听到
- 挂断结束通话

### 2.2 TCP三次握手

```
客户端                    服务器
  │                        │
  │──── SYN ────────────→  │  第一次：我要连接
  │                        │
  │←─── SYN+ACK ────────── │  第二次：好的，我准备好了
  │                        │
  │──── ACK ────────────→  │  第三次：收到，开始通信
  │                        │
```

### 2.3 TCP四次挥手

```
客户端                    服务器
  │                        │
  │──── FIN ────────────→  │  我要关闭连接
  │                        │
  │←─── ACK ────────────── │  知道了
  │                        │
  │←─── FIN ────────────── │  我也要关闭
  │                        │
  │──── ACK ────────────→  │  好的，再见
  │                        │
```

## 三、UDP协议

### 3.1 UDP特点

- **不可靠**：不保证数据到达
- **无连接**：直接发送，不需要建立连接
- **无序**：数据可能乱序到达
- **面向数据报**：有消息边界

**比喻：** UDP就像寄明信片
- 直接寄出，不管对方收没收到
- 可能丢失，可能乱序
- 但速度快，开销小

### 3.2 TCP vs UDP 对比

| 特性 | TCP | UDP |
|------|-----|-----|
| 连接 | 面向连接 | 无连接 |
| 可靠性 | 可靠 | 不可靠 |
| 速度 | 较慢 | 较快 |
| 应用场景 | 文件传输、网页 | 视频直播、游戏 |

## 四、Socket编程基础

### 4.1 什么是Socket？

Socket（套接字）是网络通信的端点，就像电话的听筒。

```
服务器Socket              客户端Socket
    │                         │
    │←── 建立连接 ──────────→│
    │                         │
    │←── 发送/接收数据 ─────→│
    │                         │
```

### 4.2 Socket编程流程

**服务器端：**
```
socket() → bind() → listen() → accept() → recv()/send() → close()
 创建       绑定       监听       接受连接    收发数据      关闭
```

**客户端：**
```
socket() → connect() → send()/recv() → close()
 创建       连接        收发数据       关闭
```

## 五、TCP Socket编程

### 5.1 TCP服务器

```cpp
#include <iostream>
#include <cstring>
#include <unistd.h>
#include <arpa/inet.h>
#include <sys/socket.h>

int main() {
    // 1. 创建Socket
    int serverSocket = socket(AF_INET, SOCK_STREAM, 0);
    if (serverSocket == -1) {
        std::cerr << "创建Socket失败" << std::endl;
        return 1;
    }
    
    // 2. 设置地址结构
    sockaddr_in serverAddr;
    serverAddr.sin_family = AF_INET;           // IPv4
    serverAddr.sin_addr.s_addr = INADDR_ANY;   // 监听所有IP
    serverAddr.sin_port = htons(8888);         // 端口8888
    
    // 3. 绑定地址
    if (bind(serverSocket, (sockaddr*)&serverAddr, sizeof(serverAddr)) == -1) {
        std::cerr << "绑定失败" << std::endl;
        close(serverSocket);
        return 1;
    }
    
    // 4. 开始监听
    if (listen(serverSocket, 5) == -1) {
        std::cerr << "监听失败" << std::endl;
        close(serverSocket);
        return 1;
    }
    
    std::cout << "服务器启动，等待连接..." << std::endl;
    
    // 5. 接受客户端连接
    sockaddr_in clientAddr;
    socklen_t clientLen = sizeof(clientAddr);
    int clientSocket = accept(serverSocket, (sockaddr*)&clientAddr, &clientLen);
    
    if (clientSocket == -1) {
        std::cerr << "接受连接失败" << std::endl;
        close(serverSocket);
        return 1;
    }
    
    std::cout << "客户端连接: " << inet_ntoa(clientAddr.sin_addr) << std::endl;
    
    // 6. 收发数据
    char buffer[1024];
    while (true) {
        // 接收数据
        int bytesRead = recv(clientSocket, buffer, sizeof(buffer), 0);
        if (bytesRead <= 0) break;
        
        buffer[bytesRead] = '\0';
        std::cout << "收到: " << buffer << std::endl;
        
        // 发送回应
        const char* response = "收到你的消息了！";
        send(clientSocket, response, strlen(response), 0);
    }
    
    // 7. 关闭连接
    close(clientSocket);
    close(serverSocket);
    
    return 0;
}
```

### 5.2 TCP客户端

```cpp
    int bytesRead = recv(clientSocket, buffer, sizeof(buffer), 0);
    buffer[bytesRead] = '\0';
    
    std::cout << "收到服务器回应: " << buffer << std::endl;
    
    // 5. 关闭连接
    close(clientSocket);
    
    return 0;
```

## 六、UDP Socket编程

### 6.1 UDP服务器

```cpp
#include <iostream>
#include <cstring>
#include <unistd.h>
#include <arpa/inet.h>
#include <sys/socket.h>

int main() {
    // 1. 创建UDP Socket
    int serverSocket = socket(AF_INET, SOCK_DGRAM, 0);
    if (serverSocket == -1) {
        std::cerr << "创建Socket失败" << std::endl;
        return 1;
    }
    
    // 2. 设置地址
    sockaddr_in serverAddr;
    serverAddr.sin_family = AF_INET;
    serverAddr.sin_addr.s_addr = INADDR_ANY;
    serverAddr.sin_port = htons(9999);
    
    // 3. 绑定
    bind(serverSocket, (sockaddr*)&serverAddr, sizeof(serverAddr));
    
    std::cout << "UDP服务器启动，端口9999" << std::endl;
    
    // 4. 接收和发送数据
    char buffer[1024];
    sockaddr_in clientAddr;
    socklen_t clientLen = sizeof(clientAddr);
    
    while (true) {
        // 接收数据（UDP不需要建立连接）
        int bytesRead = recvfrom(serverSocket, buffer, sizeof(buffer), 0,
                                  (sockaddr*)&clientAddr, &clientLen);
        
        buffer[bytesRead] = '\0';
        std::cout << "收到: " << buffer << std::endl;
        
        // 发送回应
        const char* response = "UDP Echo";
        sendto(serverSocket, response, strlen(response), 0,
               (sockaddr*)&clientAddr, clientLen);
    }
    
    close(serverSocket);
    return 0;
}
```

## 七、HTTP协议

### 7.1 HTTP请求格式

```
GET /index.html HTTP/1.1
Host: www.example.com
User-Agent: Mozilla/5.0
Accept: text/html

```

**组成部分：**
- 请求行：`方法 URL 协议版本`
- 请求头：键值对
- 空行
- 请求体（POST时有）

### 7.2 HTTP响应格式

```
HTTP/1.1 200 OK
Content-Type: text/html
Content-Length: 1234

<html>...</html>
```

**常见状态码：**
| 状态码 | 含义 |
|--------|------|
| 200 | 成功 |
| 301 | 永久重定向 |
| 404 | 未找到 |
| 500 | 服务器错误 |

### 7.3 简易HTTP服务器

```cpp
#include <iostream>
#include <cstring>
#include <unistd.h>
#include <arpa/inet.h>
#include <sys/socket.h>

void handleRequest(int clientSocket) {
    char buffer[4096];
    recv(clientSocket, buffer, sizeof(buffer), 0);
    
    std::cout << "收到请求:\n" << buffer << std::endl;
    
    // 构造HTTP响应
    const char* response = 
        "HTTP/1.1 200 OK\r\n"
        "Content-Type: text/html\r\n"
        "Content-Length: 52\r\n"
        "\r\n"
        "<html><body><h1>Hello from C++ HTTP Server!</h1></body></html>";
    
    send(clientSocket, response, strlen(response), 0);
    close(clientSocket);
}

int main() {
    int serverSocket = socket(AF_INET, SOCK_STREAM, 0);
    
    // 允许端口复用
    int opt = 1;
    setsockopt(serverSocket, SOL_SOCKET, SO_REUSEADDR, &opt, sizeof(opt));
    
    sockaddr_in serverAddr;
    serverAddr.sin_family = AF_INET;
    serverAddr.sin_addr.s_addr = INADDR_ANY;
    serverAddr.sin_port = htons(8080);
    
    bind(serverSocket, (sockaddr*)&serverAddr, sizeof(serverAddr));
    listen(serverSocket, 5);
    
    std::cout << "HTTP服务器启动: http://localhost:8080" << std::endl;
    
    while (true) {
        sockaddr_in clientAddr;
        socklen_t clientLen = sizeof(clientAddr);
        int clientSocket = accept(serverSocket, (sockaddr*)&clientAddr, &clientLen);
        
        handleRequest(clientSocket);
    }
    
    close(serverSocket);
    return 0;
}
```

**测试：** 浏览器访问 `http://localhost:8080`

## 八、Linux Socket → Qt网络编程（重点！）

### 8.1 网络编程对比表

| Linux API | Qt对应类 | 说明 |
|-----------|---------|------|
| socket() | QTcpSocket/QUdpSocket | 套接字创建 |
| bind() | QTcpServer::listen | 绑定端口 |
| listen() | QTcpServer | 监听连接 |
| accept() | QTcpServer::nextPendingConnection | 接受连接 |
| connect() | QTcpSocket::connectToHost | 连接服务器 |
| send()/recv() | QTcpSocket::write/readData | 数据收发 |
| close() | QTcpSocket::close | 关闭连接 |

### 8.2 Qt TCP编程完整示例

#### Qt TCP服务器

```cpp
#include <QTcpServer>
#include <QTcpSocket>
#include <QDebug>

class TcpServer {
private:
    QTcpServer* server;
    QList<QTcpSocket*> clients;
    
public:
    TcpServer() {
        server = new QTcpServer;
        
        // 监听所有地址的8888端口
        if (server->listen(QHostAddress::Any, 8888)) {
            qDebug() << "服务器启动，端口8888";
        } else {
            qDebug() << "服务器启动失败:" << server->errorString();
        }
        
        // 新连接信号
        connect(server, &QTcpServer::newConnection, this, &TcpServer::newClient);
    }
    
    void newClient() {
        QTcpSocket* client = server->nextPendingConnection();
        clients.append(client);
        
        qDebug() << "新客户端连接:" << client->peerAddress().toString() 
                 << ":" << client->peerPort();
        
        // 接收数据
        connect(client, &QTcpSocket::readyRead, this, &TcpServer::readData);
        
        // 客户端断开
        connect(client, &QTcpSocket::disconnected, this, &TcpServer::clientDisconnected);
    }
    
    void readData() {
        QTcpSocket* client = qobject_cast<QTcpSocket*>(sender());
        if (!client) return;
        
        QByteArray data = client->readAll();
        qDebug() << "收到客户端数据:" << data;
        
        // 回复客户端
        client->write("收到你的消息: " + data);
    }
    
    void clientDisconnected() {
        QTcpSocket* client = qobject_cast<QTcpSocket*>(sender());
        if (client) {
            qDebug() << "客户端断开:" << client->peerAddress().toString();
            clients.removeAll(client);
            client->deleteLater();
        }
    }
    
    // 广播消息给所有客户端
    void broadcast(const QString& message) {
        for (QTcpSocket* client : clients) {
            client->write(message.toUtf8());
        }
    }
    
    ~TcpServer() {
        server->close();
        for (QTcpSocket* client : clients) {
            client->deleteLater();
        }
    }
};
```

### 8.3 Qt UDP编程完整示例

```cpp
#include <QUdpSocket>
#include <QDebug>

class UdpCommunication {
private:
    QUdpSocket* socket;
    
public:
    UdpCommunication() {
        socket = new QUdpSocket;
        
        // 绑定端口
        if (socket->bind(QHostAddress::Any, 9999)) {
            qDebug() << "UDP绑定端口9999成功";
        }
        
        // 接收数据
        connect(socket, &QUdpSocket::readyRead, this, &UdpCommunication::readPendingDatagrams);
    }
    
    void sendDatagram(const QString& message, const QString& host, quint16 port) {
        QByteArray data = message.toUtf8();
        socket->writeDatagram(data, QHostAddress(host), port);
    }
    
    void readPendingDatagrams() {
        while (socket->hasPendingDatagrams()) {
            QByteArray datagram;
            datagram.resize(socket->pendingDatagramSize());
            
            QHostAddress sender;
            quint16 senderPort;
            
            socket->readDatagram(datagram.data(), datagram.size(),
                                &sender, &senderPort);
            
            qDebug() << "收到UDP数据:" << datagram 
                     << "来自:" << sender.toString() << ":" << senderPort;
        }
    }
    
    // 广播消息
    void broadcastMessage(const QString& message) {
        QByteArray data = message.toUtf8();
        socket->writeDatagram(data, QHostAddress::Broadcast, 9999);
    }
    
    ~UdpCommunication() {
        socket->close();
    }
};
```

### 8.4 Qt HTTP编程

```cpp
#include <QNetworkAccessManager>
#include <QNetworkRequest>
#include <QNetworkReply>
#include <QDebug>

class HttpClient {
private:
    QNetworkAccessManager* manager;
    
public:
    HttpClient() {
        manager = new QNetworkAccessManager;
    }
    
    // GET请求
    void get(const QString& url) {
        QNetworkRequest request(QUrl(url));
        QNetworkReply* reply = manager->get(request);
        
        connect(reply, &QNetworkReply::finished, this, [=]() {
            if (reply->error() == QNetworkReply::NoError) {
                QByteArray data = reply->readAll();
                qDebug() << "GET响应:" << data;
            } else {
                qDebug() << "GET错误:" << reply->errorString();
            }
            reply->deleteLater();
        });
    }
    
    // POST请求
    void post(const QString& url, const QByteArray& data) {
        QNetworkRequest request(QUrl(url));
        request.setHeader(QNetworkRequest::ContentTypeHeader, "application/json");
        
        QNetworkReply* reply = manager->post(request, data);
        
        connect(reply, &QNetworkReply::finished, this, [=]() {
            if (reply->error() == QNetworkReply::NoError) {
                QByteArray response = reply->readAll();
                qDebug() << "POST响应:" << response;
            } else {
                qDebug() << "POST错误:" << reply->errorString();
            }
            reply->deleteLater();
        });
    }
    
    // 下载文件
    void downloadFile(const QString& url, const QString& savePath) {
        QNetworkRequest request(QUrl(url));
        QNetworkReply* reply = manager->get(request);
        
        connect(reply, &QNetworkReply::finished, this, [=]() {
            if (reply->error() == QNetworkReply::NoError) {
                QFile file(savePath);
                if (file.open(QIODevice::WriteOnly)) {
                    file.write(reply->readAll());
                    file.close();
                    qDebug() << "文件下载完成:" << savePath;
                }
            }
            reply->deleteLater();
        });
        
        // 进度
        connect(reply, &QNetworkReply::downloadProgress, this, 
                [=](qint64 bytesReceived, qint64 bytesTotal) {
            qDebug() << "下载进度:" << bytesReceived << "/" << bytesTotal;
        });
    }
    
    ~HttpClient() {
        delete manager;
    }
};
```

### 8.5 实际Qt项目中的应用场景

#### 场景1：多人聊天室

```cpp
#include <QTcpServer>
#include <QTcpSocket>
#include <QList>

class ChatServer : public QTcpServer {
private:
    QList<QTcpSocket*> clients;
    
protected:
    void incomingConnection(qintptr socketDescriptor) override {
        QTcpSocket* socket = new QTcpSocket;
        socket->setSocketDescriptor(socketDescriptor);
        clients.append(socket);
        
        connect(socket, &QTcpSocket::readyRead, this, &ChatServer::broadcastMessage);
        connect(socket, &QTcpSocket::disconnected, this, &ChatServer::clientLeft);
    }
    
    void broadcastMessage() {
        QTcpSocket* sender = qobject_cast<QTcpSocket*>(QObject::sender());
        QByteArray message = sender->readAll();
        
        for (QTcpSocket* client : clients) {
            if (client != sender && client->state() == QAbstractSocket::ConnectedState) {
                client->write(message);
            }
        }
    }
    
    void clientLeft() {
        QTcpSocket* socket = qobject_cast<QTcpSocket*>(QObject::sender());
        clients.removeAll(socket);
        socket->deleteLater();
    }
};
```

#### 场景2：网络文件传输

```cpp
#include <QTcpSocket>
#include <QFile>

class FileTransfer {
public:
    // 发送文件
    static void sendFile(QTcpSocket* socket, const QString& filePath) {
        QFile file(filePath);
        if (!file.open(QIODevice::ReadOnly)) return;
        
        // 先发送文件名和大小
        QFileInfo info(filePath);
        QByteArray header = info.fileName().toUtf8() + "|" + 
                           QString::number(info.size()).toUtf8() + "\n";
        socket->write(header);
        
        // 发送文件内容
        QByteArray data = file.readAll();
        socket->write(data);
        
        file.close();
    }
    
    // 接收文件
    static void receiveFile(QTcpSocket* socket, const QString& savePath) {
        QByteArray header = socket->readLine();
        QStringList parts = QString(header).split("|");
        QString fileName = parts[0];
        qint64 fileSize = parts[1].toLongLong();
        
        QFile file(savePath + "/" + fileName);
        if (file.open(QIODevice::WriteOnly)) {
            QByteArray data = socket->read(fileSize);
            file.write(data);
            file.close();
        }
    }
};
```

## 十二、常见面试题与解答

### 1. TCP和UDP的区别？

**答：**
- TCP：面向连接、可靠、有序、面向字节流，适合文件传输、网页浏览
- UDP：无连接、不可靠、无序、面向数据报，适合视频直播、在线游戏

### 2. TCP三次握手和四次挥手的过程？

**答：**
三次握手：客户端SYN → 服务器SYN+ACK → 客户端ACK
四次挥手：客户端FIN → 服务器ACK → 服务器FIN → 客户端ACK

### 3. 什么是Socket？

**答：**
Socket是网络通信的端点，是IP地址+端口号的组合。通过Socket可以建立连接并收发数据。

### 4. TIME_WAIT状态的作用？

**答：**
TIME_WAIT状态确保：
- 延迟的数据包可以被接收
- 防止旧连接的数据包影响新连接
- 通常持续2*MSL（最大段生命周期）

### 5. 粘包和拆包问题如何解决？

**答：**
粘包：多个小消息被合并成一个
拆包：一个大消息被分成多个
解决方法：
- 使用固定长度的消息
- 在消息前添加长度字段
- 使用特殊分隔符
- Qt中可以使用QDataStream自动处理

### 6. HTTP和HTTPS的区别？

**答：**
- HTTP：明文传输，端口80，不安全
- HTTPS：加密传输（TLS/SSL），端口443，安全
