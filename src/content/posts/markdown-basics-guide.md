# Markdown 基础语法入门指南

Markdown 是一种轻量级标记语言，它允许人们使用易读易写的纯文本格式编写文档。由于其简洁和高效，Markdown 已成为程序员、博客作者和文档撰写者的首选工具。

---

## 1. 标题 (Headings)

使用 `#` 号来表示标题，`#` 的数量代表标题的级别（1-6级）。

```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```

## 2. 字体格式 (Typography)

Markdown 支持粗体、斜体、删除线等常用格式：

- **粗体**：使用 `**文本**` 或 `__文本__`。
- *斜体*：使用 `*文本*` 或 `_文本_`。
- ***粗斜体***：使用 `***文本***`。
- ~~删除线~~：使用 `~~文本~~`。

## 3. 列表 (Lists)

### 无序列表
使用 `*`、`+` 或 `-` 后加空格：
- 项目一
- 项目二
- 项目三

### 有序列表
使用数字加英文句点和空格：
1. 第一项
2. 第二项
3. 第三项

## 4. 链接与图片 (Links & Images)

### 链接
格式：`[链接文字](链接地址)`
例如：[访问我的 GitHub](https://github.com)

### 图片
格式：`![替代文字](图片地址)`
例如：`![示例图片](https://via.placeholder.com/150)`

## 5. 引用 (Blockquotes)

使用 `>` 号来表示引用：

> 这是一个引用的例子。
> Markdown 让写作变得简单。

## 6. 代码 (Code)

### 行内代码
使用反引号包裹：`console.log("Hello")`

### 代码块
使用三个反引号包裹，并可以指定语言：

```javascript
function greet() {
  console.log("Hello, Markdown!");
}
```
